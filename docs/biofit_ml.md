# BIOFIT 서비스를 위한 ML 설계 설명

## 1. 웨어러블 디바이스 생체 데이터기반 수면 질 예측
### 1.1 catboost 모델 학습
- 서비스 회차가 지날 수록 데이터 확보를 통한 예측 고도화.
### 1.2 다양한 경우의 수를 대입하여 수면의 질을 가장 높이는 운동 시간대와 수면 시간대 도출 
- 운동 시간대와 수면 시간대 pair로 다양한 케이스 입력 후 결과 도출
## 2. 모델이 예측값을 도출한 원인 분석 (shap)
## 3. 회원 코칭 (LLM 코칭 메시지 생성): 앞서 도출된 결과와, 데이터를 LLM에 전달
### 3.1 지난 기간 동안의 생체 활동 추이 분석 
### 3.2 수면의 질 향상을 위한 운동 시간대 및 수면 시간대 추천
- 회원 운동 가능한 시간대(운동, 수면) 범위를 주고 다시 예측
## 4. 2회차 이상 회원 코칭 (LLM 코칭 메시지 생성): 1,2와 함께, 지난주 추천 내용도 함께 LLM에 전달 
- 지난 운동 수면 시간대 추천 기반으로 수면의 질 변화 추이 분석
- 수면의 질 낮은 추이를 보이면 이를 반영하여 다른 운동 시간대 추천
